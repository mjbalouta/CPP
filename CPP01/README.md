# CPP01

## EX00
<b>Main Problem</b>: Determine in which case it is better to allocate zombies on the stack or the heap.

## EX01
<b>Main Problem</b>: Learn to use new and delete for creating and deleting an horde of Zombies.

## EX02
<b>Main Problem</b>: Demystify references. Work with a string, a pointer to that string and a reference to that string. Understand the syntax to display the memory address of each one and also its value.

## EX03
<b>Main Problem</b>: Understanding when to use a pointer to the Weapon or a reference to the Weapon. Understanding the differences between the two classes that we need to create.

## EX04
<b>Main Problem</b>: Understanding string manipulation in C++ and its differences from C. Using file manipulation functions.

## EX05
<b>Main Problem</b>: Understanding pointers to member functions.

## EX06
<b>Main Problem</b>: Using the switch statement to display a list of complaints in cascade.
